## ✨ 项目简介

Alian 是一款完全开源的 AI 手机助手，支持语音对话、视频通话、屏幕共享和自动化操作，让 AI 成为你的贴心伙伴。

**它不仅仅是一个自动化工具，更是一个真正的 AI 伙伴** —— 你可以用语音和它对话，让它看懂你的屏幕或周围环境，甚至可以随时打断它正在执行的任务，像和人一样交流。

### 为什么选择 Alian？

- 🚀 **零门槛使用** - 无需电脑、无需数据线、无需 Root，安装即可使用
- 🧠 **真正的智能** - 基于视觉语言模型（VLM），能"看懂"屏幕、理解语境
- 🗣️ **自然交互** - 语音对话、视频通话、屏幕共享，多种方式与 AI 交流
- 🎭 **深度定制** - 语音/视频通话支持自定义 Prompt，打造专属 AI 角色
- 🔌 **能力可扩展** - 通过 MCP 协议连接外部工具，按需扩展 AI 能力
- 🔒 **隐私安全** - API Key 本地加密存储，敏感操作需用户确认
- 💯 **完全开源** - 代码完全开源，可自由定制和二次开发

### 核心能力

| 能力 | 描述 |
|------|------|
| 🤖 **自动化操作** | 语音指令让 AI 自动操作手机，点外卖、打车、购物等 |
| 📞 **语音通话** | 像打电话一样与 AI 聊天，支持自定义角色、打断、连续对话 |
| 📹 **视频通话** | 开启摄像头，AI 看见你的世界，可定制角色、识别物品、解答疑问 |
| 📱 **手机通话** | AI 看见你的屏幕，语音指挥 AI 操作手机 |
| 🔌 **MCP 扩展** | 自定义添加 MCP Server，连接天气、地图、搜索等外部服务 |
| ⚙️ **Prompt 定制** | 自由定制 AI 角色、性格、说话风格，每类通话可独立设置 |

## 🚀 快速体验

### 📱 移动端

1. **下载** - 前往 [Releases](../../releases) 页面下载最新版本 APK
2. **安装** - 安装 APK 到你的 Android 设备
3. **配置 API Key** - 打开应用，进入设置页面，配置通义千问 API Key

## ⚠️ 注意事项

### 权限与功能对应关系

Alian 的不同功能需要不同的权限支持。请在使用前确保已授予相应权限，否则相关功能将无法正常使用。

> 💡 **提示**：前往「设置」→「权限」页面查看和管理所有权限状态

| 权限名称 | 语音通话 | 手机通话 | 视频通话 | 手机自动化 | 说明 |
|:-------:|:-------:|:-------:|:-------:|:-------:|:-----|
| 录音权限 | ✅ 必需 | ✅ 必需 | ✅ 必需 | ✅ 必需 | 用于语音输入和语音识别 |
| 相机权限 | ❌ 不需要 | ❌ 不需要 | ✅ 必需 | ❌ 不需要 | 用于视频通话时采集画面 |
| 悬浮窗权限 | ✅ 必需 | ✅ 必需 | ✅ 必需 | ✅ 必需 | 用于显示通话界面和执行进度 |
| 无障碍服务 | ❌ 不需要 | ✅ 必需 | ❌ 不需要 | ✅ 必需 | 用于自动化操作和控制其他应用 |
| 屏幕录制权限 | ❌ 不需要 | ✅ 必需 | ❌ 不需要 | ✅ 必需 | 用于截取屏幕内容进行视觉识别 |
| Shizuku 权限 | ❌ 不需要 | ⚡ 可选 | ❌ 不需要 | ⚡ 可选 | 提供更高效的设备控制（与无障碍服务二选一） |
| 通知权限 | ⚡ 推荐 | ✅ 必需 | ⚡ 推荐 | ✅ 必需 | 用于显示后台运行通知，防止应用被系统回收 |

> **图例说明**：
> - ✅ 必需：该功能必须拥有此权限才能正常使用
> - ⚡ 可选/推荐：拥有此权限可获得更好的体验
> - ❌ 不需要：该功能不依赖此权限

### 权限配置建议

根据你的使用场景，推荐以下权限配置方案：

| 使用场景 | 推荐配置 |
|:-------|:-------|
| 仅使用语音/视频通话 | 录音权限 + 相机权限（视频通话需要）+ 悬浮窗权限 |
| 仅使用手机自动化 | 录音权限 + 悬浮窗权限 + 无障碍服务 + 屏幕录制权限（或 Shizuku 权限） |
| 使用全部功能 | 授予所有权限，获得完整体验 |

## ⚡ 实时中断与交互

### 核心亮点

Alian 最大的创新在于：**在 AI 自动执行任务的过程中，你可以随时通过语音打断它，与 AI 进行对话，并下达新的命令。**

这就像有一个真正的智能助手在帮你操作手机，你可以随时和它交流，改变主意，或者添加新的需求。

### 使用场景示例

#### 场景 1：点外卖时改变主意

```
你：帮我点个披萨
Alian：好的，正在美团上帮你点披萨...

（Alian 开始在美团上搜索披萨）

你：等一下，我改主意了，点个汉堡吧
Alian：好的，那我帮你点汉堡。你想吃什么口味的？
你：要辣味的，加个可乐
Alian：没问题，正在帮你下单辣味汉堡和可乐...

（Alian 重新在美团上搜索汉堡，选择辣味汉堡，添加可乐，下单）
```

#### 场景 2：导航时添加途经点

```
你：导航到天安门广场
Alian：好的，正在打开高德地图导航到天安门广场...

（Alian 打开高德地图，开始导航）

你：等一下，先帮我加个途经点，我想路过星巴克买杯咖啡
Alian：好的，我来帮你添加途经点。附近有家星巴克，我帮你加进去？
你：好的，就这家
Alian：已添加星巴克作为途经点，继续导航...

（Alian 修改导航路线，添加星巴克途经点，继续导航）
```

#### 场景 3：购物时询问建议

```
你：帮我买双运动鞋
Alian：好的，正在淘宝上帮你搜索运动鞋...

（Alian 打开淘宝，搜索运动鞋，浏览商品）

你：等一下，这双鞋怎么样？你觉得好看吗？
Alian：（看着屏幕）这双鞋是白色的，设计比较简约，适合日常穿着。不过如果你想要更时尚一点的话，我可以帮你看看其他款式？
你：不用了，就这双吧
Alian：好的，正在帮你下单...

（Alian 继续完成购买流程）
```

#### 场景 4：发消息时临时添加内容

```
你：给我妈妈发条微信，说我想她了
Alian：好的，正在打开微信...

（Alian 打开微信，找到妈妈，准备发送消息）

你：等一下，再加一句，问她周末要不要一起吃饭
Alian：好的，我来修改一下消息内容："妈妈，我想你了。周末要不要一起吃饭？"可以吗？
你：可以，发送吧
Alian：好的，正在发送...

（Alian 修改消息内容并发送）
```

#### 场景 5：看视频时切换内容

```
你：打开 B 站看热门视频
Alian：好的，正在打开 B 站...

（Alian 打开 B 站，开始播放热门视频）

你：等一下，我不想看这个，帮我搜一下"猫咪搞笑视频"
Alian：好的，我来帮你搜索猫咪搞笑视频...

（Alian 搜索"猫咪搞笑视频"，开始播放相关视频）
```

### 敏感操作检测

Alian 会自动检测以下敏感操作，并在执行前暂停等待确认：

| 操作类型 | 检测方式 | 处理方式 |
|---------|---------|---------|
| **支付页面** | CaptchaDetector 识别支付相关关键词 | 暂停 + 语音播报 + 用户确认 |
| **密码输入** | 检测密码输入框 | 暂停 + 语音播报 + 用户确认 |
| **验证码** | 检测验证码输入框 | 暂停 + 语音播报 + 等待用户输入 |
| **转账操作** | 检测转账相关关键词 | 暂停 + 语音播报 + 用户确认 |
| **删除操作** | 检测删除确认对话框 | 暂停 + 语音播报 + 用户确认 |
| **权限请求** | 检测系统权限请求 | 暂停 + 语音播报 + 用户确认 |

#### API Key 加密存储
- **AES-256-GCM 加密** - API Key 使用军用级加密存储
- **密钥隔离** - 每个用户的密钥独立加密
- **安全存储** - 存储在 Android Keystore 中
- **永不上传** - API Key 永远不会上传到服务器

### 用户控制权

Alian 始终将控制权交给用户：

- ✅ **随时停止** - 悬浮窗提供停止按钮，可随时停止任务
- ✅ **实时监控** - 实时语音播报当前操作
- ✅ **人工确认** - 敏感操作需要用户明确确认
- ✅ **语音中断** - 可随时语音打断并下达新命令
- ✅ **透明可见** - 所有操作都可见，无隐藏行为

## 🚀 快速开始

### 环境要求

- **Android 8.0 (API 26)** 或更高版本
- **WiFi 网络**
- **VLM API Key** - 需要视觉语言模型的 API 密钥（推荐阿里云通义千问）
- **权限方式（二选一）**：
  - **Shizuku** - 用于获取系统级控制权限（推荐）
  - **无障碍服务 + 屏幕录制** - 无需额外工具，但功能受限

### 安装步骤

#### 1. 安装 Alian

从 [Releases](../../releases) 页面下载最新 APK 安装。

#### 2. 选择权限方式（二选一）

**方式一：使用 Shizuku（推荐，功能完整）**

Shizuku 是一个开源工具，可以让普通应用获得 ADB 级别的权限，无需 Root。

- [Google Play](https://play.google.com/store/apps/details?id=moe.shizuku.privileged.api)
- [GitHub Releases](https://github.com/RikkaApps/Shizuku/releases)

**启动 Shizuku（二选一）：**

**无线调试（推荐，需 Android 11+）**
1. 进入 `设置 > 开发者选项 > 无线调试`
2. 开启无线调试
3. 在 Shizuku App 中选择"无线调试"方式启动

**电脑 ADB**
1. 手机连接电脑，开启 USB 调试
2. 执行：`adb shell sh /storage/emulated/0/Android/data/moe.shizuku.privileged.api/start.sh`

**授权 Alian**
1. 打开 Shizuku App
2. 在应用列表中找到 Alian 并授权

**方式二：使用无障碍服务 + 屏幕录制（无需额外工具）**

1. 打开 Alian App
2. 进入 `设置 > 无障碍`
3. 找到「Alian」并开启无障碍服务
4. 在 Alian App 中授予屏幕录制权限

> ⚠️ 注意：此方式功能受限，某些操作可能无法完成

#### 3. 配置 API Key

**⚠️ 重要：进入设置页面，配置你的 API Key**

### 获取 API Key

**阿里云通义千问（推荐国内用户）**
1. 访问 [阿里云百炼平台](https://bailian.console.aliyun.com/)
